# musicat
http://www.musicat.life/main

# 프로젝트 목적
- 사용자가 자신의 자작곡 혹은 멜로디를 공유하고, 음원을 활용하여 재창조한 음원을 서로 공유하며, 음악을 좋아하고 즐기는 사람들의 소통과 발전을 위해 관련 커뮤니티 역할에 대한 필요성으로 프로젝트를 수립

# 프로젝트 기대 효과
- 사용자가 공유한 음원 청음 가능
- 신인 프로듀서, 작곡가들의 커뮤니티 형성 및 의견 공유
- 장르 혹은 태그에 따라 상세한 정보 조회 가능
- 자신만의 플레이리스트 설정 가능
